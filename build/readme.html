Einfach Ambulant WebPortal Prototype¶

Installation¶

  1. Install Docker and Docker-Compose
  2. Clone this repository
  3. Configure the environment variables in docker-compose.yml
  4. Run docker-compose up
  5. Go to http://localhost:5000

Releasing a new version¶

  1. Improve the Code
  2. Build a new Docker image with docker build -t einfachambulant/webportal .
  3. Push the image to Docker Hub with docker push einfachambulant/webportal:latest

Deploying¶

See Installation but run docker pull einfachambulant/webportal:latest on the server first.

Known Limitations¶

The web portal currently works with the following browsers, although some may not support all features.

  • Chrome 40+
  • Edge 14+
  • Firefox 40+
  • Opera 29+
  • Safari 10.3+ (iOS)

The following browsers are currently not supported.

  • Internet Explorer (all versions)
